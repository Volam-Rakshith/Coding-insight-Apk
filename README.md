# Coding Insight 💡

**By VR-Developments (Volam Rakshith Developments)**

> "See the logic behind the syntax."

Coding Insight is a powerful AI-powered code explanation & debugging assistant for Android. Paste your code, tap "Explain Code", and instantly understand what's happening — errors, explanations, and fixes all in one place!

---

## ✨ Features

- 🐛 **Smart Error Detection** — Finds syntax & logic mistakes instantly
- 🧠 **Code Explanation** — Explains what your code does in simple terms
- 🔎 **Line-by-line Insights** — Pinpoints exactly where issues occur
- 🛠 **Automatic Fixes** — Generates corrected code instantly
- 📋 **Copy Fixed Code** — One tap to copy the improved version
- 📚 **Code History** — Stores and searchs past debugging sessions
- 🌙 **Dark Mode** — Clean, developer-friendly interface
- 🔐 **No Login Required** — Private and lightweight

---

## 📥 How to Install

> Since this app is not on the Play Store yet, follow these steps:

1. Download the APK from the [Releases](../../releases) section
2. Once downloaded, tap the APK file
3. A warning popup will appear — tap **"Settings"**
4. You will be taken to **"Install unknown apps"**
5. Find your browser or file manager (e.g. Chrome, Files)
6. Toggle **"Allow from this source"** ✅
7. Press back and tap the APK file again
8. Tap **"Install"**
9. Once installed, tap **"Open"**
10. Open **Coding Insight** and start coding! 🚀

> ⚠️ **Note:** This is completely safe! You can turn off
> "Allow from this source" after installation for security.
